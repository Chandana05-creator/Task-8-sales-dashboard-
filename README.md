
# 📊 Task 8 – Simple Sales Dashboard Design

## 📌 Objective
Build an interactive sales dashboard using Power BI to analyze sales performance by month, region, and category. The dashboard helps business users understand sales trends through visualizations and filters.

---

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Superstore Sales Dataset (CSV)

---

## 📂 Dataset
- Dataset Name: Superstore Sales
- File Format: CSV

---

## 📈 Dashboard Visualizations
The dashboard includes:

- 📉 Line Chart – Sales by Month-Year
- 📊 Bar Chart – Sales by Region
- 🍩 Donut Chart – Sales by Category
- 🎛️ Slicer – Region
- 🎛️ Slicer – Category

---

## 🔄 Data Preparation
- Imported the CSV dataset into Power BI.
- Converted the **Order Date** into **Month-Year** format using DAX.

```DAX
Month-Year = FORMAT([Order Date], "MMM yyyy")
```

---

## 📊 Key Insights
1. West region generated the highest sales.
2. South region recorded the lowest sales.
3. Technology category contributed the highest sales.
4. Sales varied across different months, showing changing business performance over time.



## 🎯 Outcome
This project demonstrates how to:
- Import and transform data in Power BI.
- Create interactive dashboards.
- Visualize sales trends using different chart types.
- Use slicers for dynamic filtering.
- Generate business insights from data.
